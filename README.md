# pix
Rust crate for pixel and raster images.
Currently a work-in-progress.

## Documentation
[https://docs.rs/pix](https://docs.rs/pix)

## Future Plans
* Gradient (linear, radial, etc) and GradientIter

### Color models / spaces
* Tristimulus, additive, subtractive, cylindrical
* RGB: additive
* CMYK: subtractive
* HSV / HSL: cylindrical
* LAB, LCH
* XYZ: tristimulus
* White point
